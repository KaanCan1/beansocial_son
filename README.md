# ☕ BeanSocial

> A social platform for coffee lovers — share recipes, save your favorite brews, and discover new types of coffee.

![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?logo=dart&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?logo=firebase&logoColor=black)
![GetX](https://img.shields.io/badge/GetX-8A2BE2)

BeanSocial is a cross-platform mobile app built with Flutter as my Computer Engineering graduation project. Users can create and share coffee recipes, build a personal collection of favorites, and explore different coffee types in a clean, social feed.

## ✨ Features

- 🔐 **Authentication & profiles** — sign up / sign in with Firebase Authentication
- 📝 **Recipes** — create, share and browse coffee recipes
- ⭐ **Favorites** — save brews and build a personal collection
- 🔄 **Real-time data** — synced with Cloud Firestore
- 🧭 **Discovery feed** — explore new coffee types
- ⚙️ **State management** — handled app-wide with GetX

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Flutter |
| Language | Dart |
| State | GetX |
| Backend | Firebase (Auth + Cloud Firestore) |

## 🚀 Getting Started

```bash
# clone the repo
git clone https://github.com/KaanCan1/beansocial_son.git
cd beansocial_son

# install dependencies
flutter pub get

# run on a connected device / emulator
flutter run
```

> A Firebase project is required. Add your `google-services.json` (Android) and/or `GoogleService-Info.plist` (iOS) before running.
